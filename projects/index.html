FIRST Robotics Simulator
(Open Source, C++) January 2010-Ongoing

Code: Google Code Repository

Project: Windows Program that emulates the laws of Physics and the FIRST WPI Library to allow our robot's code to run on a computer when physical access to the GNU-based robot and its hardware was impossible to obtain. I worked on the field rendering, as well as the porting of FIRST code to Windows

Logomotion Score Calculator
(OpenSource, Javascript) January 2011

A javascript score calculator for the 2011 FIRST Robotics game logomotion.

Naive Bayesian Sentiment Analysis of LiveJournal
(Private, Python) December 2008

For my science fair project as a sophomore in high school, I trained multiple naive bayes on approximately 3.5 million livejounal posts tagged with the top 100 emotions. Once the models were trained, they tried to guess the emotion of 1000 other posts they were not trained on.
